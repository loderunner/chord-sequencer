Class: Sequence

Sequence

A Sequence encapsulates a complete song. It contains global attributes, like song key or tempo. It also contains a Collection of Chord events.

Constructor

new Sequence()

Properties:
Name Type Description
key string

The (tonal) key of the song. Defaults to 'C'.
mode string

The mode of the scale. Defaults to 'Major'.
tempo number

The tempo of the song in BPM. Defaults to 120.
loopLength string

The length of the current loop in Tone.js musical time notation. Defaults to '1m'.
chordList ChordList

The actual list of Chord events in the sequence.
