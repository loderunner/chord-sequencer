Class: Chord

Chord

A Chord encapsulates a complete song.

Constructor

new Chord()

Properties:
Name Type Description
step string

The degree (or step) of the root note of the chord in the scale in roman numerals. Defaults to 'I'.
seventh boolean

true if the chord contains a seventh note. Defaults to false.
start string

The start time of the chord in the sequence, in Tone.js time notation. Defaults to '0m'.
duration string

The duration of the chord, in Tone.js time notation. Defaults to '16n'.
